StorLever是什么？

StorLever是部署在Linux上基于python的配置、管理、监控系统，用于管理服务器上的各种网络和存储资源（功能）。它提供了一组精心设计的RESTful风格的可编程接口（API），以及一个简介的Web操作面板。 更多详细 »

为什么storlever？

随着云计算的流行普及，管理员需要同时管理越来越多的虚拟主机。而StorLever的出现，将帮助管理员轻而易举地从困难、痛苦、而又耗时的各种配置管理工作中解脱出来。它结合如openfiler、freeNas、Ajenti、OpenLMI等类似开源项目的优点，是一个易用、可拓展、轻量级的解决方案。 更多详细 »

StorLever可接管的资源

  • Block device
  • SCSI device
  • MD Raid
  • Mega Raid (LSI chip, through extension)
  • LVM
  • XFS
  • EXT4
  • NFS Client & Server
  • SAMBA Server
  • FTP Server
  • iSCSI Initiator & Target(TGT)
  • SMART
  • Ethernet Interface
  • Bond Interface
  • Route Table
  • DNS Config
  • NTP
  • SNMP
  • User&Group
  • Host
  • Zabbix Agent
  • Mail Agent
  • HTTP

StorLever的特色

  • 集成了Linux系统的多种存储管理技术，包括LVM，阵列，NAS/SAN等等
  • 提供多种操作接口，包括RESTful风格API，Web UI，CLI（另外项目），SDK（另外项目）
  • 可扩展，添加插件十分容易
  • 简单，纯Python
